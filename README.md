# [trevorcorc.com](https://trevorcorc.com)

## What is this?

This site is dedicated to all things me. I have no idea
what I plan on putting here, so please check back every so
often to see what's new.

## Technologies Used

This is a static website hosted on netlify (because it's free!) that
uses the `Vue` framework along with the material design framework
provided by `vuetify`.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
